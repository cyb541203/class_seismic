01. Integration method: Newmark-gamma Method¶

In [1]:
clear, clc, close ALL





















Equation of motion¶













\begin{equation*}
\ddot{u(t)} + 0.5919 \dot{u(t)} + 39.4384 u(t) = f(t)
\end{equation*}











In [2]:



    


M = [1];
C = [0.5919];
K = [39.4384];



    















    
























In [3]:



    


F = 9.81 * load('tohoku_acc.txt');



    















    
























In [4]:



    


dt = 0.01;
[N, nd] = size(F); %30000*1 matrix



    















    
























In [5]:



    


X  = zeros(N,nd);
Xv = zeros(N,nd);
Xa = zeros(N,nd);



    















    
























In [6]:



    


% Integrating methods: Newmark beta method
%% Average acceleration (or trapezoidal rule) / Implicit

gamma = 0.5;
beta  = 0.25;

a0    = 1     / ( beta * dt^2 );
a1    = gamma / ( beta * dt   );
a2    = 1     / ( beta * dt   );
a3    = 1     / ( beta * 2    )-1;
a4    = gamma / beta - 1;
a5    = dt    / 2 * ( gamma / beta - 2 );
a6    = dt    * ( 1 - gamma );
a7    = gamma * dt;

Ke    = K + a0*M + a1*C;
Ke_inv= inv(Ke);



    















    
























In [7]:



    


for i=2:N
    Feff    = F(i,:)' + M * (a0 * X(i-1,:)' + a2 * Xv(i-1,:)' + a3 * Xa(i-1,:)');
    Feff    = Feff    + C * (a1 * X(i-1,:)' + a4 * Xv(i-1,:)' + a5 * Xa(i-1,:)');
    X(i,:)  = (Ke_inv * Feff)';
    Xa(i,:) = a0 * (X(i,:)-X(i-1,:)) - a2 * Xv(i-1,:) - a3 * Xa(i-1,:);
    Xv(i,:) = Xv(i-1,:) + a6 * Xa(i-1,:) + a7 * Xa(i,:);
end



    















    
























In [8]:



    


resp = [(0.0:dt:dt*(N-1))', X, Xv, Xa];
save response.txt resp -ascii
